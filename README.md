## KTH IS1200 PROJECT
### BY FREDRIK EKLÖÖF and SIMON ÖSTERGREN

## DESCRIPTION
This is a RFID Card Reader that signals an analog servo to rotate clockwise or counter clockwise
depending on whether the lock should open or close.

## Hardware
* ChipKit Uno32
* Futaba S3003
* RFID MFRC-552
